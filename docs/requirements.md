# IssueHub Requirements

## Overview

IssueHub is a full-stack Issue Tracking application built with:

- FastAPI
- SQLAlchemy
- PostgreSQL (Production)
- SQLite (Development)
- React
- TypeScript
- Vite
- Railway
- Vercel

---

## Authentication

- Register
- Login
- JWT Authentication
- Logout
- Current User
- Change Password

---

## User Profile

- View Profile
- Upload Avatar
- Replace Avatar
- Delete Previous Avatar Automatically

---

## Issue Management

- Create Issue
- Edit Issue
- Delete Issue
- View Issue Details

---

## Filtering

- Search
- Status Filter
- Priority Filter

---

## Security

- JWT Protected Routes
- Password Hashing
- Ownership Validation

---

## Deployment

Backend:
- Railway

Frontend:
- Vercel

- PostgreSQL

Storage:
- Local Uploads (Development)
